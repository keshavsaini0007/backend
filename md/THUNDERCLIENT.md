# ⚡ Thunder Client - Quick Overview

**Thunder Client** is a lightweight API testing tool built directly inside VS Code.

---

## 🚀 What It Does

* Send HTTP requests (GET, POST, PUT, DELETE)
* Test APIs without leaving your editor
* View responses instantly

---

## 🛠️ Why Use It

* Fast and simple
* No need for external tools like Postman
* Beginner-friendly interface

---

## 💡 Example

```
GET http://localhost:3000/api/users
```

---

## 📦 Key Features

* Request collections
* Environment variables
* Auth support (JWT, Basic Auth)
* Clean JSON response view

---

## 🔥 One Line Summary

**Thunder Client = API testing inside VS Code, fast and minimal.**
